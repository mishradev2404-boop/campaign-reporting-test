Connect the dashboard to your GitHub repository. The dashboard will automatically fetch and display the latest CSV report whenever it loads.

Go to your CSV file on GitHub → click Raw → copy that URL. It looks like: raw.githubusercontent.com/…
Format: https://api.github.com/repos/{owner}/{repo}/contents/{path}
Generate at GitHub → Settings → Developer settings → Personal access tokens → Fine-grained. Needs Contents: Read permission. Stored in your browser only.

Upload a CSV file directly from your computer. You'll need to re-upload each time you want to update the data.
